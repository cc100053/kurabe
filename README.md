# kurabe

Flutter app for comparing prices with smart camera and shop lookup.

## Environment setup
1. Copy `.env.example` to `.env`.
2. Fill `GOOGLE_PLACES_API_KEY` with your Google Places API key.
3. Run with `flutter run` (the env file is loaded on startup via `flutter_dotenv`).
